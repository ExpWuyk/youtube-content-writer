为 YouTube 视频生成国内平台搬运发布文案。

## 使用方式

用户提供 YouTube 视频链接，按以下流程生成全套发布文案。

## 工作流程

### 1. 提取视频元数据

运行脚本获取视频信息（请在 skill 安装目录中找到脚本）：

```bash
python "~/.claude/skills/youtube-content-writer/scripts/fetch_video_meta.py" "<youtube_url>"
```

### 2. 补充搜索（必做）

用 WebSearch 并行搜索以下信息：

- **歌词/台词/内容解析**：搜索 `"<视频标题>" 歌词/解析/考察/review`
- **热度/传播情况**：搜索 `"<视频标题>" bilibili/抖音/viral/TikTok`
- **背景信息**：搜索作者/IP/作品背景

### 3. 生成全套文案

读取 `~/.claude/skills/youtube-content-writer/references/content-prompt.md` 获取完整输出规则，严格按照其中格式输出：

**输出结构：**

```
## 第一步：视频核心信息解析
### 1. 核心定位
### 2. 高光亮点
### 3. 合规信息
### 4. 关键词库

## 第二步：全套可直接复制发布文案
### 1. 作品标题（5组，均≤30字）
### 2. 作品简介（完整版/精简版/极简版）
### 3. 话题标签（20个，带#，分层级）
```

## 硬性规则

- 标题单条 ≤ 30 字，无例外
- 所有内容必须基于视频实际内容，禁止空泛套话
- 必须包含原素材来源标注（作者、频道、发布时间）
- 规避敏感词和限流风险
- 输出纯文本发布文案，禁止生成剪辑/分镜等内容
- 全部中文输出（专有名词保留原文）
